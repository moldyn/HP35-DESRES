## HP35-DESRES
The time series of backbone dihedral angles of of Nle/Nle mutant of villin headpiece (HP35), generated from a 300Î¼s long molecular dynamics simulation (Amber99SB) at 360K.

Quick-start:
``` 
    git clone https://github.com/moldyn/HP35-DESRES.git
    cd HP35-DESRES
    bunzip2 -kt traj.dat.bz2
```
